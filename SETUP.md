# Setup guide

### Required software
##### NodeJS
https://nodejs.org/en/

##### MySQL
Easiest solution for windows users: xampp

For linux: TBD

https://www.apachefriends.org/index.html

### Installation

1. Run "npm install" in both "front-end" and "back-end" folder
2. Start xampp and turn on the MySQL server and Apache
3. Go to http://localhost/phpmyadmin
4. Create a new database called "wishhabbo"
5. In "back-end" folder, run "npm run re:migrate"

### Start up

1. One terminal in "front-end" folder and run "npm start"
2. Another terminal in "back-end" folder and run "npm run dev"

Visit http://localhost:4200
